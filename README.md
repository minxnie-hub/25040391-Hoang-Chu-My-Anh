## Chạy trên máy
Mở trực tiếp `index.html`, hoặc chạy local server:

```bash
python -m http.server 8000
```

Sau đó mở `http://localhost:8000`.

## Thay ảnh cá nhân
Thay file sau bằng ảnh mới, giữ nguyên tên file:

`assets/profile/hoang-chu-my-anh.png`

Nên dùng ảnh dọc hoặc ảnh chân dung, tối thiểu 1200 px ở cạnh dài.

## Deploy GitHub Pages
1. Đưa toàn bộ thư mục này lên repository.
2. Vào **Settings → Pages**.
3. Chọn **Deploy from a branch**.
4. Chọn branch `main`, thư mục `/ (root)`, rồi Save.

## Cấu trúc
- `index.html`: trang chủ
- `bai-1.html` … `bai-6.html`: nội dung chi tiết từng bài
- `style.css`: giao diện và responsive
- `script.js`: menu, hiệu ứng xuất hiện, mục lục và xem ảnh
- `assets/docs`: ảnh minh chứng được trích xuất theo đúng thứ tự từ bài gốc

Phần quảng cáo/download của source tham khảo đã được loại bỏ hoàn toàn.
